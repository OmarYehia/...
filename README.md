# Programming Quiz: What do I Wear? (3-7)
Using if/else statements, create a series of logical expressions that logs the size of a t-shirt based on the measurements of:
    - shirtWidth
    - shirtLength
    - shirtSleeve

    - S, M, L, XL, 2XL, or 3XL
